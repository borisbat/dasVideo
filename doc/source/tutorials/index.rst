Tutorials
=========

A ladder that builds up the dasVideo API one step at a time. The first two are
headless libraries with self-verifying tests; the last three open a window and put
the video (and sound) on screen, wrapping the runnable ``examples/``. Every step is
fronted by **Emma** — each tutorial is voiced, and she walks through it as the code
runs.
